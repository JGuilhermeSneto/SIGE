# Módulo de Comunicação

## Visão geral

O app `apps.comunicacao` gerencia comunicados, avisos e murais internos do SIGE com segmentação por público.

## Funcionalidades principais

- Criação, edição e exclusão de comunicados.
- Segmentação por tipo de usuário e público-alvo.
- Controle de publicação por data.
- Exibição de mural centralizado.

## Estrutura de pastas

- `models/` — modelo de comunicado.
- `views.py` — endpoints de gestão e listagem.
- `forms.py` — formulários para comunicados.
- `templates/` — layouts de listagem e formulário.

## Uso principal

Use este app para publicar avisos institucionais e entregá-los aos perfis corretos na plataforma.

## Observações

Os comunicados podem aparecer para alunos, professores e gestores conforme a segmentação definida.
