Sobre a API SIGE

O SIGE expõe uma API RESTful completa baseada em Django Rest Framework. Utilizamos drf-spectacular para gerar automaticamente a especificação OpenAPI 3.0, garantindo documentação sempre atualizada com o código.

OpenAPI 3.0 JWT Auth JSON Support
{# ── AUTENTICAÇÃO ── #}

Segurança e Autenticação

A maioria dos endpoints exige autenticação via JSON Web Token (JWT). Para obter um token, envie suas credenciais para o endpoint de login.

Authorization: Bearer <seu_token_jwt>

Guia Rápido de Uso

1. Obtenha o Token: Acesse o endpoint /api/token/ enviando seu nome de usuário e senha.

2. Autorize no Swagger: Clique no botão "Authorize" (cadeado) no topo do Swagger UI e insira Bearer <seu_token>.

3. Consulte a Documentação: O arquivo de especificação bruta (YAML/JSON) pode ser baixado em /api/schema/.

4. Ambientes: A API está disponível em /api/v1/ para todos os módulos principais.
