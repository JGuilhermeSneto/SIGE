Incidente: CPU em 100%

Crítico

Utilize este runbook quando o sistema apresentar lentidão extrema ou o monitor de recursos indicar saturação de processamento.

1. Diagnóstico

Acesse o servidor via SSH e utilize o comando top ou htop.

# Buscar top 5 processos gastões
ps aux | sort -rk 3,3 | head -n 5

Arquitetura do Ecossistema SIGE

1. Core Engine (Django v6.x)

O coração do sistema opera sob o padrão Service Layer Pattern, garantindo o desacoplamento total da lógica de negócio das Views e facilitando a testabilidade.

2. Stack Tecnológica

Python 3.13+ Django 6.0.3 PostgreSQL / Redis React + Vite (SPA) Celery (Async)

3. Frentes de Operação

  • Web Interface: SPA focada em alta densidade de dados com Design Glassmorphism.
  • Canal Mobile: React Native com autenticação SimpleJWT e Push Notifications.
  • Camada IoT (ESP32): Protocolo MQTT sobre WebSockets para automação física e controle de acesso.

Segurança, Compliance & SOC

1. Proteção de Dados (PII)

Implementação de criptografia robusta AES-256 GCM para todos os dados sensíveis (Personally Identifiable Information), garantindo conformidade total com a LGPD.

2. SOC & Defesa de Borda

Monitoramento ativo via SecurityShieldMiddleware. O sistema detecta e bane automaticamente atacantes que tentam acessar vetores sensíveis.

# Monitorando Honeypots em tempo real
tail -f logs/security_audit.log | grep "BLOCK_IP"

3. Backup & Resiliência (Snapshot Quantum)

Backups imutáveis e redundantes gerados automaticamente a cada ciclo de manutenção, permitindo a restauração total do ambiente em menos de 15 minutos.

Estratégia de Testes & Qualidade

1. Testes Unitários

Garantem que funções individuais e modelos funcionem conforme o esperado.

python manage.py test apps.academico.tests

2. Testes de Integração

Validam a comunicação entre módulos (ex: Financeiro gerando faturas após matrícula).

3. Cobertura de Código

Mantemos uma meta de 85% de cobertura mínima em módulos críticos.

Comandos Úteis de Administração

# Limpar Cache Total
python manage.py clear_cache

# Criar Superusuário Emergencial
python manage.py createsuperuser

# Sincronizar Estáticos via Terminal
python manage.py collectstatic --noinput

2. Ciclo Financeiro
